To run on FLIP:

1. Make sure all files are in the same folder.
2. Type 'make'. This will create a 'TSP' object.
3. To run program, type ./TSP followed by as many test files as desired

Example:
./TSP tsp_example_1.txt tsp_example_2.txt tsp_example_3.txt

This will output the results to respective files now in the same folder:
tsp_example_1.txt.tour
tsp_example_2.txt.tour
tsp_example_3.txt.tour
