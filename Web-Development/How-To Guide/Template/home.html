Intro to the NASA API

View the NASA API Homepage

An application program interface (API), is a set of routines, protocols, and tools for building software applications. In general terms, it is a library of tools of a specific type made available by a person or organization to users for their own applications. The NASA API grants users access to unique features that allow them to pull data from the NASA database.

Getting your API Key

Why use an API Key?

An API Key is a string of text unique to you that is required for your call to an API. This allows the provider of the API to track its use, as well as prevent abuse of its services. In this section we will learn why and how we will use the NASA API key and how to obtain one for use in our own applications. We will also review security guidelines for our key.

Working with JSON

Some useful JSON documentation

JSON stands for JavaScript Object Notation. JSON is commonly used in conjunction with web APIs to exchange information from its host to be used in a variety of applications. We will explore more in depth what JSON is and how you can use it to use the NASA API to gather the information you need.

Picture of the Day

What is today's POTD?

The NASA Picture of the Day API is one of the more popular APIs NASA has to offer. It returns an image and relevant properties associated with the image that can be parsed and used in a variety of ways.
