Putting it Together

See more of the Mars Rover API

Below is an example of code that one might use in their application. This code first loads the manifest data of the Curiosity rover and allows the user to select a sol date in a given range. After clicking the button to get pictures, the first three pictures from that day are displayed. The Earth date, rover name, and the camera used to take the photo are also displayed.
