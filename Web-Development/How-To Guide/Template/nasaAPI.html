Introduction to the NASA API

The NASA open APIs homepage

NASA has committed itself to providing open and full sharing of all data that it obtains with academia, private industry, the general public, and luckily for us, the research and applications communities. What does this mean for us? As programmers and application creators, this means that NASA has made available to us its vase wealth of knowledge that it has obtained. To do so, we have access to NASA's API.

What is an API?

An API is a tool, or more concisely, a library that allows developers to safely access information for developers to use in other software. For example, the OpenWeatherMap is a weather API that provides developers with information on the weather throughout the world for use in their applications. This can be highly useful if you are creating an app that utilizes a weather feature, because you do not have to create a research station to find out what the weather will be like tomorrow. Instead, you can use code provided by the API to pull information on the weather from the OpenWeatherMap servers!

APIs are also very common in social media. Many applications and websites make use of the Facebook API in order to post on behalf of users (with their permission), create photo albums, or use services like finding out if Jane is friends with Dick. Twitter, Youtube, Google, Yahoo, and many other large companies have free and and easily accessible APIs. These are great for you as a developer, great for the user of your application as they can use a more familiar service embedded in your app (in the case of popular social media APIs) and great for the companies that provide the API because it increases the value of their product, drives revenue in many instances, and can increase the popularity of their business. For educational and research based organizations, the free exchange of information is also an incredibly appealing aspect of APIs.

What does the NASA API do?

NASA has several open source APIs that allow developers to access their data in useable ways.

NASA Open APIs

  • Picture of the Day: New image everyday with useful keywords
  • Asteroids - NeoWs: Lookup specific asteroid or search based on closest approach date to Earth
  • EPIC: Daily imagery of the EPIC instrument. Full disc imagery of the Earth
  • EONET: Browse the entire globe daily in near real time
  • Earth: Satellite imagery powered by Google Earth Engine
  • Mars Rover Photos: Images gathered by the Mars Rovers
  • Patents: NASA patent portfolio
  • Sounds: A collection of sounds in space!
  • SSD/CNEOS: Interface to machine-readable data realted to SSD and CNEOS
  • Techport: Data for technology projects NASA is working on right now
In this guide we will focus on the Picture of the Day and the Mars Rover Photo portions of the NASA API. In the next section, we will find out how to obtain an API Key for use in our application, and how to use the API to get the information we need from the NASA server to our application.
