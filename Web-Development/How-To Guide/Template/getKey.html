Getting your API Key

To apply for a NASA API key apply here.

So far we have explored what an API is and some of the things it might be used for, including many of the open APIs from NASA. In order to make the most use of these APIs as a developer, we must obtain an API Key. But what is an API Key and why do we need to use it?

The API Key

An API Key is a unique "key" assigned to you and you alone for use with your application. In most cases, you will be unable to access the data services of an API without including an API key issues by the content creator. A key serves as a vital piece of security and tracking tool for the company that issues it. It can be used as a form of identification and authentication for a service to identify where a request is coming from. Many companies limit the amount of requests you can make to their server within a certain time period before your application is "throttled" or blocked. The API key makes this type of tracking possible, in the case of an abuse of their services.

Another use of the API Key that benefits the developer, is being able to more easily manipulate your code and keep it up to date. For example, if a web service providing you with an API instead required a username/password and you needed to change your password down the road, you would then need to change every instance of your password in your code which is cumbersome and prone to error. Overall, an API Key is more secure for you as a developer because it is extremely difficult to guess, and a leaked key is less likely to give away information about you than a normal username/password combination would.

How do I get an API Key?

The process for obtaining an API Key for the NASA APIs is thankfully very straightforward. Navigate to the NASA API Key application form and fill in your details. You will then receive your API key immediately and it will be also emailed to you. Remember to keep your key secure at all times, especially in your code.
